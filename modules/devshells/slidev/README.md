# Slidev Development Environment

## Quick Start

```bash
nix develop .#slidev
```

## Available Tools

-   **Node.js 22**: JavaScript runtime
-   **pnpm**: Fast package manager

## Usage Examples

```bash
# Create a new Slidev project
pnpm create slidev

# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Export to PDF
pnpm export
```
